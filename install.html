Minx!_OS — Advanced Installation Guide

This guide is for advanced users who want to build Minx!_OS from the ground up, similar to Arch Linux or Linux From Scratch. No graphical installer is provided.

1. Boot the Live Environment

Boot into the Minx!_OS live ISO and open a terminal.

2. Connect to the Internet

# Ethernet
ip link

# Wi-Fi
iwctl

3. Partition the Disk

Use fdisk or cfdisk:

cfdisk /dev/sda

Create:

  • EFI partition (300MB, FAT32)
  • Root partition (all remaining space, ext4)

4. Format the Partitions

mkfs.fat -F32 /dev/sda1
mkfs.ext4 /dev/sda2

5. Mount Target System

mount /dev/sda2 /mnt
mkdir /mnt/boot
mount /dev/sda1 /mnt/boot

6. Install the Base System

(Replace with your actual package manager — pacstrap equivalent)

minx-install base linux linux-firmware

7. Generate fstab

genfstab -U /mnt >> /mnt/etc/fstab

8. Chroot Into the New System

minx-chroot /mnt

9. Set Timezone, Locale, Hostname

ln -sf /usr/share/zoneinfo/America/Chicago /etc/localtime
hwclock --systohc

nano /etc/locale.gen
locale-gen

echo "minx-os" > /etc/hostname

10. Install and Configure the Bootloader

minx-install grub efibootmgr
grub-install --target=x86_64-efi --efi-directory=/boot --bootloader-id=MinxOS
grub-mkconfig -o /boot/grub/grub.cfg

11. Set Root Password

passwd

12. Reboot

exit
umount -R /mnt
reboot

13. (Optional) Install Desktop Environment

Examples:

minx-install plasma
minx-install gnome
minx-install xfce4

Congratulations — you have successfully built Minx!_OS from scratch.
